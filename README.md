# minimal-d3d12-imgui-hook

Similar to https://github.com/Sh0ckFR/Universal-Dear-ImGui-Hook but cleaned up a bit and using [vcpkg](https://github.com/microsoft/vcpkg#quick-start-windows) for dependency management.


### Disclaimer

This is a personal project I use as a general starting point when I want to create an imgui overlay on a dx12 game. I will not provide support using it or fix any issues you might encounter w/ it.
